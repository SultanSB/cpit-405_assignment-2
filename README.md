# Nationality Predictor - Assignment 2
This project uses the Nationalize.io API to predict the possible nationalities of a person based on their name.
The user enters a name, and the app displays a list of countries with their associated probabilities.

## How to Run
1. Download or clone the project.
2. Open index.html in any browser
3. Type a name and press Submit - results will appear instantly!


### Example Output
```
US - Probability: 45.12%
GB - Probability: 20.43%
IN - Probability: 12.76%
```
